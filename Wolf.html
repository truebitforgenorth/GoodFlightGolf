🐺 Wolf Rules

Game Logic

How Scoring Works:

  • Team holes use the Base value.
  • Lone, Dump, and Blind each use fixed point amounts that you set below.
  • Win and Lose values are separate, so you can customize each outcome however you want.
  • Fixed solo wins pay exactly the amount you enter — no extra math is added.
  • If a pot/carryover exists, it is only used when points need to be split between multiple winners.
  • This game does not use negative points — only winners receive points.

Dump Lose 1 vs 3 The other 3 players Each winning player gets your set Dump Lose Points Pot is split evenly between the 3 winners
Blind Win 1 vs 3 The Blind Wolf The Blind Wolf gets your set Blind Win Points No split math — pays the exact Blind Win amount you entered
Blind Lose 1 vs 3 The 3 opponents Each winning opponent gets your set Blind Lose Points Pot is split evenly between the 3 winners
🔁 Push / Carryover
Situation What Happens
Push / Tie No player gets points for that hole.
Carryover Enabled Your set Tie Carryover Points are added to the pot.
Future Win with Pot The winners for that hole receive both their normal points and the carryover pot.
