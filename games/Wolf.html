🐺 Wolf Rules

Game Logic

How Payouts Work:

  • Team holes use the Base value that you set.
  • Lone, Dump, and Blind each use their own custom win and lose values.
  • When a hole has multiple winners, the payout is split evenly among those winners unless that scoring mode says otherwise.
  • When there is a solo winner, that player receives the full value for that scoring mode.
  • If carryover is enabled, tie points are added to the pot and paid out on a later winning hole.
  • This section explains how the app pays out points or money, not the playing rules of Wolf.

🔁 Tie / Carryover Payouts
Situation Payout Result
